# CORDEX-CMIP5
METACLIP bindings of CORDEX-CMIP5 Controlled vocabularies 

## Installation

To clone this repository using SSH:

```bash
git clone git@github.com:metaclip/CORDEX-CMIP5.git
```

Alternatively, you can clone using HTTPS:

```bash
git clone https://github.com/metaclip/CORDEX-CMIP5.git
```
